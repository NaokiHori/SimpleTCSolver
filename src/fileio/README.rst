#######
fileio/
#######

Kernel functions which take care of the file input and output are implemented.

* internal.h

   Private functions which are used in this directory are declared.

* parallel.c

   Parallel file loaders and writers are included.

* serial.c

   Serial file loaders and writers are included.

* snpyio.c, snpyio.h

   See `SimpleNpyIO <https://github.com/NaokiHori/SimpleNpyIO>`_

* utility.c

   Other common functions are included.
