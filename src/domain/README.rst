#######
domain/
#######

Information about the spatial domain (e.g. resolution, grid points) is handled.

* finalise.c

   Destructor.

* init.c

   Allocator and loader of the parameters.

* internal.h

   Internal header for private functions.

* io.c

   Load and save parameters in ``domain_t``.

* optimise.c

   Pencil domain decomposition optimiser for three-dimensional domains.
